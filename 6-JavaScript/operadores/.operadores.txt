/Operadores matematicos/Ariméticos de comparación
/*
- MAYOR QUE (.. > ..)
- MENOR QUE  (.. < ..)
- MAYOR O IGUAL (.. >= ..)
- MENOR O IGUAL (.. <= ..)
- IGUAL (.. == ..)
- DISTINTO (.. != ..)
*/

/*
1) 25 >= 40 -------> False
2) 25 >= 25 -------> True
3) "hola" == 20.4 -----> False
4) "hola" == "hola"  ----> True
5) "hola" != 20.4 ------> True 
*/

 Condicion1 || Condicion 2

*/
